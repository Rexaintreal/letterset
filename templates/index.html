How it works

01

Draw each letter

Open the canvas and draw A to Z, numbers and punctuation using your mouse or stylus. One character at a time.

02

We trace your strokes

Each drawing is converted to a vector glyph automatically, No scanning, no printing, no lighting issues.

03

Download your font

When you're done, hit build and get a real .ttf file you can install on any computer.
